GET.api/v1.1

This call returns API version.

GET.api/v1.1
Returns
String: current API version.
Example
https://stellarfox.net/api/v1.1

POST.api/v1.1/auth

This call issues JWT (JSON Web Token) based on user credentials.

POST.api/v1.1/auth
Parameters
email (String) matching valid email format
password (String) matching valid password format
Returns
Object: authorization JSON including JWT.
Example
https://stellarfox.net/api/user@example.com/P@33w0r$

POST.api/v1.1/user/update

This call updates user attributes passed as query string parameters. The update will only happen if valid JWT is provided.

POST.api/v1.1/user/update
Parameters
token (String) JSON Web Token (JWT) issued per user.
Example
